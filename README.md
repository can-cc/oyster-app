## generate launcher icon
[flutter_launcher_icons](https://pub.dev/packages/flutter_launcher_icons).


``` yaml

  flutter_launcher_icons: "^0.7.2"

flutter_icons:
  image_path: "assets/logo.png"
  android: "launcher_icon"
  ios: true

```
